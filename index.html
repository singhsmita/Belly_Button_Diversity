Project Overview

This project collected a demographic information of bacteria data from volunteers' belly button. Base on these data, we created a interactive dashboard to visulize each volunteer's belly button biodiversity.

By selecting Test Subject ID number from the dropdown menu, the individual's demographic information table and following three charts will dynamically response. The horizontal bar chart identifys the subject's top 10 bacterial species in their navel. The gauge chart reveals reveals how often does the subject clean his belly button. The bubble chart displays bacteria culture test result for each sample.
